1. PHP ย่อมาจากอะไร? 
- PHP ย่อมาจาก "PHP: Hypertext Preprocessor"

2. ใครคือผู้สร้างภาษา PHP?
- ผู้สร้างภาษา PHP คือ Rasmus Lerdorf

3. รากฐานของคำสั่งของภาษา PHP มาจากภาษาอะไรบ้าง?
- รากฐานของคำสั่งของภาษา PHP มาจากภาษา C, Perl, และ Java.

4. Xampp สามารถดาว์นโหลดได้จากเว็บไซต์ใด? 
- XAMPP สามารถดาวน์โหลดได้จากเว็บไซต์ https://www.apachefriends.org/

5. PHP เป็นโปรแกรมภาษาที่ทำงานในรูปแบบ Server Slide Script มีลักษณะเป็นอย่างไร? 
- PHP เป็นโปรแกรมภาษาที่ทำงานในรูปแบบ Server-Side Script ซึ่งหมายความว่าโค้ด PHP จะถูกประมวลผลที่เซิร์ฟเวอร์ ก่อนที่จะส่งผลลัพธ์ (เช่น HTML) กลับไปยังเบราว์เซอร์ของผู้ใช้.

6. จงยกตัวอย่างข้อดีของภาษา PHP? (ระบุเป็นข้อๆ ) 
- ง่ายต่อการเรียนรู้: มีโครงสร้างที่เข้าใจง่าย
- โอเพนซอร์ส: ไม่มีค่าใช้จ่ายในการใช้งาน
- รองรับฐานข้อมูลหลายประเภท: เช่น MySQL, PostgreSQL
- ชุมชนที่ใหญ่: มีเอกสารและซัพพอร์ตมากมาย
- ประสิทธิภาพสูง: เหมาะสำหรับการพัฒนาเว็บแอปพลิเคชัน

7. โปรแกรมจำลองเซริฟเวอร์มีอะไรบ้าง ยกตัวอย่างมา 3 ข้อพร้อมการทำงานต่างกันอย่างไร?
- XAMPP: รวม Apache, MySQL, PHP, และ Perl เพื่อใช้งานง่ายสำหรับการพัฒนา
- WAMP: มี Apache, MySQL, PHP สำหรับ Windows เท่านั้น
- MAMP: ใช้สำหรับ macOS มี Apache, MySQL, PHP และง่ายต่อการติดตั้ง
